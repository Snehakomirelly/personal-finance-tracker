# Personal Finance Tracker

A full-stack web application that helps users manage their personal finances by tracking income, expenses, budgets, savings goals, and spending habits.
##  Features

### User Authentication
- User Registration
- Secure Login System
- Logout functionality
###  Transaction Management
- Add income and expenses
- Delete transactions
- Categorize transactions
- Add notes
###  Dashboard Analytics
- Total Income
- Total Expenses
- Current Balance
- This Month Spending
- Average Daily Expense
- Top Spending Category
### Budget Management
- Set monthly budget
- Budget tracking
- Budget warning alerts
### Savings Goal
- Create savings goals
- Set target amount
- Track saved amount
- Progress percentage
###  Charts
- Expense category analysis
- Monthly spending trend chart
###  Dark Mode
- Light/Dark dashboard theme
###  Export
- Export transactions as:
  - PDF
  - CSV
###  Responsive Design
- Mobile-friendly interface
- Optimized dashboard layout
##  Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Chart.js
### Backend
- Python
- Flask
### Database
- SQLite
- SQLAlchemy
### Authentication
- Flask Login
- Werkzeug Password Hashing
